# 日報チェック PWA v0.6

日報と明細を照合するローカルPWAです。

## v0.6 変更点

- 以前のUIに近い大ボタン構成へ変更
- 黒ボタンは「日報を撮影」「明細を撮影」
- 白ボタンは「写真を選択」に短縮
- OCR貼り付け欄と使い方は折りたたみへ移動
- 画像も売上データもサーバーには送信しません

## 更新方法

ZIPを解凍して、以下をGitHubへアップロードしてください。

- index.html
- app.js
- styles.css
- manifest.webmanifest
- sw.js
- README.md
- icon-192.png
- icon-512.png

GitHub Pages反映後、古い画面が残る場合はブラウザを再読み込みしてください。
